Enigma Machine Manual

The German Enigma was a feared electro-mechanical machine theoretically capable of 3,283,883,513,796,974,198,700,882,069,882,752,878,379,955, 261,095,623,685, 444,055,315,226,006,433,615,627,409,666,933,182,371, 154,802,769,920,000,000,000 conﬁgurations (Miller, 1996).

The Enigma Machine configurations consists of the following:

    Choice and order of rotors
    The alphabet keyring setting, the position of the rotor wiring - relative to the alphabet keyrings
    The plug connections on the plugboard
    A key identifier group to identify the code book number on the key sheet

The Enigma Machine transmits messages using five letter group pairings.

To identify the message key, a five letter group must be inserted. The five letter group is composed of two random characters and then the KEY IDENTIFIER. The five letter key identifier is inserted at the beginning of the message and should not be encrypted. The key identifies which rotors are used and in which order, the setting of the alphabet keyrings and the position of the rotor wiring, and the plug connections on the plugboard as formatted in the CODE BOOK. Once the key is identified the Enigma Machine can be properly configured to decrypt the rest of the message.
