Enigma Machine Manual

The German Enigma was a feared electro-mechanical machine theoretically capable of 3,283,883,513,796,974,198,700,882,069,882,752,878,379,955, 261,095,623,685, 444,055,315,226,006,433,615,627,409,666,933,182,371, 154,802,769,920,000,000,000 conﬁgurations (Miller, 1996).

The Enigma Machine configurations consists of the following:

    Choice and order of rotors
    The alphabet keyring setting, the position of the rotor wiring - relative to the alphabet keyrings
    The plug connections on the plugboard
    A key identifier group to identify the code book number on the key sheet

The Enigma Machine transmits messages using five letter group pairings.

To identify the message key, a five letter group must be inserted. The five letter group is composed of two random characters and then the KEY IDENTIFIER. The five letter key identifier is inserted at the beginning of the message and should not be encrypted. The key identifies which rotors are used and in which order, the setting of the alphabet keyrings and the position of the rotor wiring, and the plug connections on the plugboard as formatted in the CODE BOOK. Once the key is identified the Enigma Machine can be properly configured to decrypt the rest of the message.

The following letter combinations will be used for special characters:

    PELA = Parentheses
    MM = Comma
    XX = Colon
    XY = Semicolon
    YY = Dash/Hyphen/Slant
    OO = Period
    UD = Question Mark
    ZED = Space

The following letter combinations will be used for numbers:

    NILL = Zero (0)
    ONU = One (1)
    ESOD = Two (2)
    REYS = Three (3)
    QUAD = Four (4)
    PENTA = Five (5)
    SUCA = Six (6)
    NEVAS = Seven (7)
    OCHO = Eight (8)
    INES = Nine (9)
    CENTA = Tens (00)
    MILLE = Hundreds (000)
    MYRIA = Thousands (0000)

For example, 200 = ESODM ILLE, 19:48 = ONUIN ESXXQ UADOC HO

Enigma Machine Button and Text Field Usage:

    Reflector: The reflector allows the user to change between reflectors ‘B’ and ‘C’.
    Rotors: There are 8 possible rotors for the user to choose from to place in the Enigma Machine’s 3 rotor slots.
    Ring Setting: By clicking inside any of the 3 ring setting text fields, a user has the ability to change the starting position of each corresponding rotor.
    Ground Setting: By clicking inside any of the 3 ground setting text fields, a user has the ability to generate a key identification group when wanting to transmit an encrypted message to a receiving party.
    Plugboard: By clicking inside any of the 26 plugboard text fields, a user has the ability to adjust the plugboard configurations in a manner of the users choice.
    Book: By clicking the ‘Book’ button, a user has the option to choose a preprogrammed code book setting.
    Random: By clicking the ‘Random’ button, a user has the option to randomize the entire Enigma Machine.
    Send: By clicking the ‘Send’ button, a user can send his or her state of the Enigma Machine to a receiving party in order to encrypt and decrypt messages amongst each other.
    Send Clear Text: By clicking the ‘Send Clear Text’ button, sends the message that the user has typed into the clear text text-field to the server. All unencrypted messages are in red font color.
    Send Cipher Text: By clicking the ‘Send Cipher Text’ button, sends the encrypted version of the clear text message. The text field for the cipher text is locked. All encrypted messages are in black font color.
